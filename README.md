
模组介绍
=======
本模组旨在为玩家提供一个简单的使用json自定义掉落物消失后的产物。玩家可以自定义配置转化条件来实现诸如：

-鸡蛋掉落物变成小鸡（或者成年鸡）实体;

-腐肉掉落物变成皮革掉落物;

-树苗掉落物变为树苗方块等功能;

## 使用方式
配置文件位于版本文件夹 `config/itemdespawntowhat/*.json` 路径下。

每个json文件中包含一个示例，包含了所有可自定义的条目，条目留空或删除会返回默认值。

json为列表的形式，可以接在已有示例后面继续增加（复制粘贴大括号中的内容，两个大括号间用逗号隔开），不要删除json中的“[]”，否则会因为无法检测到列表而崩溃。

下面以鸡蛋掉落物变成鸡为例解释如何自定义配置文件：
```json
[
{
"item": "minecraft:egg",
"dimension": "",
"need_outdoor": false,
"surrounding_blocks": {
"north": "",
"south": "",
"east": "",
"west": "",
"up": "",
"down": "minecraft:hay_block"
},
"result": "minecraft:chicken",
"conversion_time": 5,
"result_multiple": 1,
"result_limit": 30,
"entity_age": 0
}
]
```

| 字段名                  | 说明                                                       |
|----------------------|----------------------------------------------------------|
| `Item`               | 定义转化的掉落物注册名                                              |
| `dimension`          | 掉落物进行转化的维度，留空默认为全维度                                      |
| `need_outdoor`       | 是否需要露天                                                   |
| `surrounding_blocks` | 六面的方块。注意检测的是掉落物所在格子的六面，半砖上可能对半砖无效                        |
| `result`             | 结果的注册名。不同的 json 文件负责解析为不同的类型，注意不要填错文件（名字写错或留空不会生效也不会报错）  |
| `conversion_time`    | 转化时间（单位：秒，不会超过 300 秒）                                    |
| `result_multiple`    | 一个物品可以转化为几个                                              |
| `result_limit`       | 结果数量限制。默认检测范围为掉落物周围半径 `6` 格的正方体。若结果是掉落物，限制数量为该数字 * `64`个 |
| `entity_age`         | 如果是存在年龄的实体可以进行设置，例如幼年动物可以设置为 `-2400` （注意：小僵尸不属于年龄实体的范畴）  |
