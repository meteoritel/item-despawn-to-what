# NeoForge 1.21.1 API notes

- `PlayerList.broadcastSystemMessage(Component message, boolean bypassHiddenChat)` broadcasts a system message to all online players.
- `PlayerList.broadcastSystemMessage(Component message, Function<ServerPlayer, Component> playerMessageFactory, boolean bypassHiddenChat)` can provide per-player variants.
- `ServerPlayer.sendSystemMessage(Component message)` sends a system/chat message to one player; `sendSystemMessage(Component message, boolean overlay)` is the overlay variant.
- `StreamCodec.unit(instance)` is the standard codec pattern for zero-field custom payloads.
